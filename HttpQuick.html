HttpQuick

コマンドラインからhttpファイルを読み込みHTTPリクエストを簡単に行うツールです。HTTPファイルを指定し、リクエストを実行します。

対象ユーザー: コマンドラインユーザー、サーバーにアクセスしてテストを行いたい人

開発背景: 頼まれたので作成した自動化ツールです。

バッチファイル等のスクリプトを用意し実行することも可能です。

使用方法

以下は簡単な使用例です。



                    ファイル:test.http
                    GET {{access_url}}/user/userlist/json
                    ###
                    GET {{access_url}}/user/data
                    Name: Alice
                    Data: {{data_type}}
                


                    ファイル:.env.json
                    {
                        "access_url": "localhost"
                        "data_type": "Age"
                    }
                


                    コンソール上
                    HttpQuick test.http

{{トークン名}}としておくと、.env.jsonから読み込んでリプレスして解釈します。

主要なコマンドや機能:

  • 実行例: HttpQuick test.http
  • 別のフォルダに出力(先にフォルダを作成してください): HttpQuick test.http ./response
  • ファイル名に日付を追加(出力先指定とは順不同): HttpQuick test.http -d ./response
  • ヘルプを表示: HttpQuick -h

インストール方法

---!注意! いかなる損害についても、私は責任を負いません。自己責任でご利用ください。---

ツールはPyPIからインストールできます。


                    pip install httpquick

PyPIリンク: https://pypi.org/project/httpquick/
